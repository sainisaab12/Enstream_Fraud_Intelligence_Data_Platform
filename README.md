# EnStream Fraud Intelligence Platform

A carrier-backed trust and fraud intelligence solution that processes transaction events from major telecommunication providers (Bell, Rogers, Telus) in real-time, applying an advanced rule-scoring engine to flag suspicious behaviors (e.g., SIM swapping, device-churning fraud rings, blacklisted activities).
